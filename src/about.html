About Call to Roll

Roll dice with friends, anywhere in the world

What is Call to Roll?

Call to Roll is a free, open-source web application that lets you roll dice with friends in real-time. Whether you're playing tabletop RPGs, board games, or just need a fair way to make decisions, Call to Roll provides a simple and reliable solution.

No accounts, no downloads, no hassle. Just create a room, share the code, and start rolling.

Key Features

  • Multiple dice types: d4, d6, d8, d10, d12, d20, and d100
  • Customizable sets: Create up to 4 dice sets with different colors
  • Real-time sync: Everyone sees rolls instantly
  • Roll history: Track all rolls with timestamps and player names
  • Works everywhere: Desktop, tablet, or mobile - no app required

How It Works

Call to Roll uses peer-to-peer technology (WebRTC) to connect players directly to each other. This means your dice rolls are synchronized instantly between all players without going through a central server. The result is a fast, reliable, and private gaming experience.

Our signaling server only helps players find each other to establish the initial connection. Once connected, all game data flows directly between players.

Perfect For

  • Tabletop RPGs: D&D, Pathfinder, Call of Cthulhu, and more
  • Board games: Any game that needs dice rolling
  • Remote gaming: Play with friends across the world
  • Quick decisions: Fair random number generation
  • Classroom activities: Games and probability lessons
