Getting Started

Creating a Room

  1. Enter your name in the text field
  2. Click "Create Room"
  3. Configure your dice sets:
    • Choose the number of dice (1-10 per set)
    • Select colors for each set
    • Add up to 4 different sets
    • Enable "Allow dice locking" if desired
  4. Click "Start Room"
  5. Share the 4-digit room code with your friends

Joining a Room

  1. Enter your name in the text field
  2. Click "Join Room"
  3. Click the dice to set the room code (each die cycles 1-6)
  4. Click "Enter Room" when the code matches

Rolling Dice

Once in a room, you'll see the dice sets configured by the room creator. To roll:

  1. Click on a dice set to "grab" it (your name appears above)
  2. Click the dice set again to roll
  3. Everyone in the room sees your roll in real-time
  4. Click another set or area to "drop" the dice

Only one player can hold a dice set at a time. If someone else has grabbed the dice, wait for them to roll or drop them.

Dice Locking

If the room has dice locking enabled, you can set aside specific dice between rolls (useful for games like Yahtzee):

  1. After rolling, click on individual dice to lock them
  2. Locked dice show a green border and lock icon
  3. When you roll again, locked dice keep their values
  4. Click a locked die to unlock it

Room Codes

Room codes are 4-digit numbers where each digit is 1-6 (like dice faces). This gives 1,296 possible combinations. Room codes are displayed as interactive dice that you click to cycle through values.

Frequently Asked Questions

DiceBox uses peer-to-peer connections, so the practical limit depends on your network. Typically, 6-10 players work well. For larger groups, some players may experience connection issues.

Connection issues are usually caused by:
- Firewall or network restrictions blocking WebRTC
- VPN software interfering with connections
- The room code being incorrect
Try disabling VPN, checking your firewall settings, or verifying the room code.

Browser Support

DiceBox works best on modern browsers:

  • Chrome / Edge (recommended)
  • Firefox
  • Safari (iOS 15+ / macOS 12+)
  • Opera

Internet Explorer is not supported.
